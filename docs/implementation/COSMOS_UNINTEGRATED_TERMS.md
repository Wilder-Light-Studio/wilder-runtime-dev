# Cosmos — Unintegrated Terms Ledger

A minimal, stable ledger of metaphysical terms coined during the Cosmos design work that are not yet integrated into the formal runtime docs (REQUIREMENTS / SPEC / PLAN).

These are conceptual primitives and safety-net notes only — they do not change runtime requirements.

## Metaphysical Primitives

- **blip** — atomic temporal projection key
- **smile** — semantic value associated with a blip
- **record** — metaphysical truth unit (distinct from runtime ledger entry)
- **truth** — the metaphysical substrate; the world as it *is*
- **projection** — the act of emitting a blip
- **semantic value** — the meaning carried by a smile

## Metaphysical Structures

- **RECORD** (capital-R) — the metaphysical structure composed of blips and smiles
- **truth graph** — the metaphysical structure of relationships between RECORDs/blips
- **projection key** — identity of a blip
- **temporal atom** — an indivisible unit of time (blip-level)

## Metaphysical Behaviors

- **truth projection** — metaphysical emission of a blip/projection
- **truth perception** — metaphysical awareness (distinct from runtime `Perception`)
- **truth ordering** — metaphysical temporal sequence

## Purpose

- **Single source** for concepts not yet slotted into formal docs
- **Preserve** emerging metaphysical vocabulary from being lost in working memory
- **Serve** as a migration map when unifying metaphysics and runtime documentation
